Overview

Platformer is an educational endeavor focusing on 2D physics and gameplay mechanics, featuring basic enemies and compatibility with both keyboard and controllers. Designed for learning and enjoyment, it provides a fun platform to grasp fundamental concepts of 2D game development.

Additional Details

Following the completion of TankGame, I delved into exploring advanced concepts such as 2D physics and particle systems during pandemic time. Eager to expand my skill set, I embarked on a series of learning projects to apply newly acquired knowledge effectively. Platformer emerged as a demonstration of these concepts, incorporating features like particle systems, 2D physics, and various player actions such as jumping, double jumping, dashing and wall jumping. The project also introduced elements like camera follow and basic enemy interactions, offering a comprehensive learning experience in 2D game development.

Key Features

  • Fluid Movement: Enjoy smooth and responsive controls for seamless gameplay.
  • Jumping Abilities: Master various jumping techniques, including double jumps and wall jumps.
  • Enemies: Conquer challenges posed by basic enemies, adding depth to gameplay.
  • Level Editor: Create custom levels with the built-in editor for endless replay value.
